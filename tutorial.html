Konfigurasi di MikroTik

  1. Pastikan port API MikroTik sudah aktif [ip->services].
  2. Setting jam dan tanggal dengan menambahkan NTP Serer [system->sntp client].
  3. Sesuaikan zona waktu masing-masing [system->clock].

Mikhmon + Webserver

Tutorial Singkat Mikhmon + Webserver

  1. Download dan extract [mikhmonv3ws.zip].
  2. Jalankan [MikhmonServer] / Run as administrator.
  3. Klik Open Mikhmon atau buka http://127.0.0.1 di browser (disarankan menggunakan Google Chrome).
  4. Login menggunakan user password default [user:mikhmon pass:1234].
  5. Klik Add Router
  6. Isikan IP, user dan password MikroTik, kemudian save dan klik Connect untuk tes koneksi ke MikroTik.
    [IP bisa menggunakan ip yang di ether1, cek dari Winbox : ip->address->ether1].
  7. Buat User Profile baru, atau update User Profile lama dengan memilih Expired Mode. Sesuikan dengan kebutuhan.
  8. User Profile sudah siap, selanjutnya sudah bisa add user atau generate user dan cetak voucher.

Pembuata User Hotspot

Validity = Masa Aktif, waktu akan terus berjalan dipakai mautun tidak dipakai sejak login.
Time lImit = Durasi, batas penggunaan semalam masa aktif, waktunya akan berhenti jika tidak dipakai sejak login.
Jika ingin membuat user dengan validity 30 hari, sukup buat user profile dengan validity 30d. Kemudian generate adatu add user dengan user profile tersebut. Time limit tidak perlu diisi.
Pembuatan user dengan validity dan time limit idealnya time limit lebih kecil dari validity.
Contoh membuat user dengan time limit 7 hari dan masa aktif 30 hari, maka bisa add user adau generate user dengan time limit 7d dan memilih user profile yang memiliki validity 30d.

Perhatikan protocol yang digunakan https / http.
Untuk server local yang menggunakan PC atau Android biasanya menggunakan protocol http.
Perhatikan pada gambar bagian yang diberi tanda merah dan hijau
Untuk Session Name domain servermikhmon / IP server Mikhmon (*IP server Mikhmon adalah IP PC/Android dimana Mikhmon diinstall).
Bagi yang menggunakan Mikhmon V3, sesuaikan Session Name (lihat gambar) untuk menampilkan expired pada laman status hotspot5.
Session Name dapat dilihat di Session Settings (Settings->Session Settings). Untuk yang masih menggunkan Mikhmon V2 bisa merubah script menjadi seperti gambar berikut.
